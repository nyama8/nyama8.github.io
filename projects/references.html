Projects

reference-manager

This is a simple set of Python classes and methods for scraping article metadata for journals and conference proceedings. The resulting classes enable the creation of citations.

The scraper looks for article metadata by searching the HTML meta tags that Google Scholar uses for indexing. Some websites do not use particular set of meta tags (Highwire Press), which requires further work. Additionally, some websites block parsing by scripts, in which case the HTML sources can be downloaded and passed directly.

Module descriptions

  • reference.py: A module containing the Reference class and several methods for creating creating Reference classes. The following are examples of included objects:
    • Reference (class): A class that reprsents a single scientific work
    • reference_from_url() (function): A method that gets the reference metadata from a url
    • reference_from_src() (function): A method that gets the reference metadata from a website's html source file (necessary when websites block access via scripts).
  • scraper.py: A module containing methods for scraping metadata from journal and conference publication websites. Primarily used as a helper module to the reference.py module, but can be used for other applications without the Reference class.
